API Usage

You can also use our API to convert text to speech programmatically:

Endpoint:
POST /api/tts
Request Body (JSON):
{
  "text": "Your text to convert",
  "voice": "en-US-ChristopherNeural",
  "format": "mp3"
}
Response:
{
  "success": true,
  "file_url": "/audio/filename.mp3"
}
Get available voices:
GET /api/voices
